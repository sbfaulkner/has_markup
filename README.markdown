# has\_markup

Easily add markup support with syntax colouring to a model.

## Example

    # models/post.rb
    class Post < ActiveRecord::Base
      has_markup :body
    end

## Installation

    $ script/plugin install git://github.com/sbfaulkner/has_markup.git
